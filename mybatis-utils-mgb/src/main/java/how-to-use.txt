1，根据类路径下的generatorConfig.xml配置文件，通过DOM4J动态生成。
2，在动态生成配置文件的Table节点的时候，需要连接数据库获取所有的表名以及根据表名确定类名。
3，动态生成mybatis代码。


继承Spring使用时：
配置Configuration的bean
配置DataSource的bean
可以自定DbAccess组件，继承自DbAccess即可。默认是用MySQL查询数据库所有表名。
配置XmlConfigBuilder的Bean，2参构造和3参构造（自定义DbAccess组件实现）。
